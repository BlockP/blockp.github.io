## 安装方法  
对下载后的文件右键，选择“解压到当前文件夹”  

![](./a.png)   
打开解压的文件夹，并代开内部的文件夹，打开其中的“Plain Craft Launcher 2”。  

![](./s1.png)   
随后，点击“下载”或“下载游戏”。  

![](./s2.png)   

在“下载”中，选择最新正式版本，或在“正式版”中选择你需要的版本，这里以 1.20.2 版本为例。  

![](./s3.png)  

选中版本后，点击“开始安装”。    

![](./s4.png)   

接下来，等待安装完成。  

![](./s5.png)   

安装完成后，点击“游戏用户名”，随便输入一个名字，然后点击“开始游戏”。  

![](./s6.png)
